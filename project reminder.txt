🛒 ویژگی‌های اصلی یک سایت فروشگاهی (حداقل نسخه)

برای شروع، یه MVP (کمینه محصول قابل ارائه) می‌زنیم که شامل این بخش‌ها باشه:

1. مدیریت کاربران (User Management)

ثبت‌نام و ورود کاربران (JWT Auth / Session)

پروفایل کاربری (نام، ایمیل، آدرس، شماره تماس و …)

فراموشی رمز عبور و تغییر رمز عبور

2. مدیریت محصولات (Products)

دسته‌بندی محصولات (Category)

محصول با فیلدهایی مثل:

نام محصول

توضیحات

قیمت

موجودی انبار

تصاویر محصول

تخفیف‌ها و کد تخفیف

3. سبد خرید و سفارش‌ها (Cart & Orders)

اضافه‌کردن محصول به سبد خرید

مدیریت سبد خرید (حذف و تغییر تعداد)

ثبت سفارش

وضعیت سفارش (در حال پردازش، ارسال شده، تکمیل شده)

4. درگاه پرداخت (Payment)

برای شروع می‌تونیم پرداخت رو شبیه‌سازی کنیم (Fake Payment)

بعداً میشه درگاه واقعی مثل زرین‌پال یا Stripe اضافه کرد

5. مدیریت پنل ادمین

اضافه، ویرایش و حذف محصولات

مدیریت کاربران

مدیریت سفارش‌ها

مشاهده گزارش فروش

6. API برای فرانت‌اند

چون احتمالاً میخوای با React یا Next.js فرانت‌اند بسازی، باید بک‌اند رو REST API بزنیم.
استفاده از Django Rest Framework (DRF) بهترین انتخابه ✅
